#Carrito de compras
Se trata de un sitio web de venta de cursos, los cuales se pueden agregar al carrito y tambien al LocalStorage para mantenerlos en el carrito aun cuando se cierra el navegador.

##Tecnologias usadas
* JavaScript
* HTML
* CSS
* Skeleton
